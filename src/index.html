Run Algorithm
About
Tiny project that I developed on spare time to learn TypeScript, SVG and as a Graph algorithms refresher. You can:
  • Drag start (blue) and destination (green) vertex around the screen and choose from 3 path finding algorithms
  • Set the block and diagonal weight
  • Add obstacles by start dragging on any empty grid block
  • Remove obstacles by start dragging on any obstacle
Every block is a graph vertex and they're connected through a visual representation. It's mobile friendly so you should be able to touch and drag the nodes around the screen on mobile devices.
